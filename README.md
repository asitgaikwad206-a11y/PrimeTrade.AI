# PrimeTrade.AI
Methodology

This analysis explores the relationship between market sentiment (Fear/Greed index) and trader behavior on Hyperliquid.

Key steps included:

Cleaning and validating both datasets
Converting timestamps to daily format
Creating trader-level daily metrics
Merging sentiment data with trader performance
Performing behavioral segmentation
Generating insights using statistical aggregation and visualization
Key Insights
Fear periods generated the highest profitability, with average daily PnL reaching 209k, significantly outperforming other sentiment regimes.
Trading activity increased sharply during Fear, with an average of 4183 trades, indicating aggressive market participation.
Neutral markets showed the weakest performance, suggesting limited trading opportunities during low-volatility conditions.
Strategy Recommendations
Increase trading activity during Fear periods, but maintain smaller position sizes to manage volatility risk.
Reduce exposure during Extreme Greed periods, as profitability declines and reversal risk increases.
